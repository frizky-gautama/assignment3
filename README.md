# Assesment3
## How to install
~~~
git clone https://github.com/bram-mncbank/assesment3
~~~

## How to Run
### Run WEB server
Enter directory and run the main.go:
~~~
cd assesment3
go run main.go
~~~
### Access the web and API
1. Open any browser and access *http://localhost:8080*
